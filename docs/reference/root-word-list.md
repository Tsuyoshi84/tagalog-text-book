# Root Word List

## Introduction

This reference lists common Tagalog root words organized by category. Root words form the foundation of Tagalog vocabulary. When combined with affixes, they create verbs, nouns, and adjectives with various meanings and grammatical functions. Each root is presented with its basic meaning and common derivatives.
